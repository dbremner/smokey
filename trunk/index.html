Smokey

Description

Smokey is a open source command line tool used to analyze .NET or Mono assemblies for problems. It's similar to tools like FxCop and Gendarme. This version of Smokey has 175 rules. Currently Smokey runs on Mono 1.2.6 or above and on .NET.

Smokey can generate reports of problems found in html, text, or xml formats. Html reports can be generated with commands like mono bin/smokey.exe -html -out:report.html -set:ignoreList:SysIgnore.txt /Users/jessejones/New_Files/mono-1.9/mcs/class/lib/net_2_1/System.Core.dll (SysIgnore.txt can be found in the source download).

Related Tools

Gendarme is the original Mono assembly checker. It was created for the 2005 Google Summer of Code but development has been slow since then and it has a limited number of rules.

FxCop is Microsoft's tool. It has a lot of rules and is a mature product. But it only runs on Windows, is not open source, and is missing some cool rules such as ConsistentEquality, CtorCallsVirtual, NullDeref, StaticSetter, and ValueHashCode.

MMC the Mono model checker is an interesting research tool that exhaustively simulates execution of assemblies and looks for problems like deadlocks and invalid assertions.

FindBugs is a Java tool. Unlike FxCop it focuses more on outright bugs then design violations.

mono-api-info is used internally by the Mono team to ensure that their public APIs do not change between releases.
